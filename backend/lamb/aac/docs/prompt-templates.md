## What are Prompt Templates?

Reusable configurations containing a system prompt and prompt template. Use them to create multiple assistants with the same base personality or instructional approach.

## Managing Templates

Access from Learning Assistants > Prompt Templates tab.

- **+ New Template** — create with name, description, system prompt, prompt template
- **Edit** — modify an existing template
- **Share / Unshare** — make available to other educators in your org
- **Duplicate** — create a copy
- **Delete** — remove a template

## Loading a Template

When creating or editing an assistant, click **Load Template** next to the System Prompt field. This populates the assistant's system prompt and prompt template from the saved template.

## Shared Templates

Toggle "Shared Templates" to see templates shared by other educators in your organization.
